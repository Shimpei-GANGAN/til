### 波としての画像
画像の「周波数」に注目する。

音は波の性質を持つ。シンセサイザー。

$$
y = \sin{(x)}
$$

``` python
import numpy as np
x = np.arange(0, 10, 0.01)
y = np.sin(x) # (1000, )
```

$$
z = \sin{(x+y)}
$$

``` python
x = np.arange(0, 10, 0.01)
y = np.arange(0, 10, 0.01)
z = np.sin(x + y) # (1000, 1000)
```

[0, 1] に出力を合わせる

$$
z = \frac{\sin{(x+y)}+1}{2}
$$
