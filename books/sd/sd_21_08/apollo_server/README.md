# Apollo Server
- GraphQLサーバサイド実装の代表
- Node.js環境で動作し、デフォルトではWebフレームワークのExpressの上で動作する

## Ready to explore your graph?
```bash
$ mkdir apollo_server
$ cd apollo_server
$ touch index.js
$ npm init --yes

$ npm install apollo-server graphql
```

実行する際
```bash
$ node index.js
```
